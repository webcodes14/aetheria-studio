# Aetheria-studio

### 🚀 Klíčové vlastnosti projektu

Multijazyčnost: Podpora CS, EN, KO přes Context API.

Barevné schéma: Tmavé a světlé schéma stránky přes Context API.

Bezpečnost: Implementace Honeypotu proti spam-botům.

Validace: Robustní validace dat pomocí knihovny Zod a React Hook Form.

UI/UX: Plynulé animace chybových hlášek díky Framer Motion.

Error Handling: Vlastní 404 a Error stránky pro lepší uživatelský zážitek.

### Použitý stack (Technologie)

* React (Vite)
* Tailwind CSS
* Framer Motion
* Zod + React Hook Form
